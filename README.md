Aplicação de simulação de deslocamento de veículos feita com NextJS e Material-UI.

## Começando

Inicie o server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu computador para ver o resultado.

[https://desloc.vercel.app/]
